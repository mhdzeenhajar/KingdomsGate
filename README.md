# Kingdom's Gate
**A C# Systems Engineering Project**

Kingdom's Gate is a 2D action-survival game developed from the ground up using **C#** and the **Raylib-cs** library. This project focuses on custom engine architecture rather than utilizing commercial game engines (like Unity or Unreal).

## 🛠️ Technical Implementation
* **State-Driven Architecture:** Implemented a robust centralized state machine to manage real-time transitions between gameplay loops and interactive tutorials.
* **Physics Systems:** Custom encumbrance logic where player velocity and jump force are calculated dynamically based on inventory mass.
* **Vector Mathematics:** Utilized `Math.Atan2` and vector navigation for the dynamic objective tracking system.

## 🚀 How to Run
1.  Ensure you have the **.NET 10.0 SDK** installed.
2.  Clone the repository: `git clone https://github.com/mhdzeenhajar/KingdomsGate.git`
3.  Navigate to the directory and run: `dotnet run`
